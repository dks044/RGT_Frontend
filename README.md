# ⚡로컬 프로젝트 실행방법 (백엔드+프론트)

## 🖼️프론트엔드
> node js 설치상태에서 하는것을 권장해요
```
--리눅스
git clone https://github.com/dks044/RGT_Frontend.git
cd RGT_Frontend
npm install
touch .env
echo "NEXT_PUBLIC_API_URL=http://localhost:8080" >> .env
npm run dev

--윈도우
git clone https://github.com/dks044/RGT_Frontend.git
cd RGT_Frontend
npm install
echo NEXT_PUBLIC_API_URL=http://localhost:8080 > .env
npm run dev
```
